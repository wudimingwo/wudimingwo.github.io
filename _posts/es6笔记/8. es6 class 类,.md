> 上面的代码到底干了什么?
> 一个类返回的对象可以通过 设置 *[Symbol.iterator](){yield} , 进行迭代
> yield *this.arr[Symbol.iterator]() 的意思是把arr的指针添加进实例对象的指针组之中.

> 测试 可以 yield * this.arr, 因为默认会调用arr的迭代器[Symbol.iterator]
```
              class iteratorArray {
                constructor (arr = []) {
                  this.arr = arr;
                }
                
                *[Symbol.iterator](){
                  yield *this.arr
                }
              }
              
              let arr = new iteratorArray([1,2,3]);
              
              for(let item of arr) {
                console.log(item);
              }
```
